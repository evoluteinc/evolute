Actions

Buttons
Button variants using .btn as the base class. Combine with .btn-primary, .btn-secondary or .btn-accent. Supports icons, disabled state and full-width layout.
Input Types
Form fields using .field with .label + .input, .select or .textarea. All share border, hover and focus styles with box-shadow. Wrapped inside a .card.
Checkbox Group
Class .checkboxes with .option wrapping each input. Layout variants: default (column), .inline (row), .grid (3 columns with CSS grid) and .pills (hidden input, pill style using :has).
